Introduction to Balance Reporting

This module is available to the user depending on the system parameters set up else the page redirects the user to the AIS link.

The GCP CashWeb Balance Reporting module offers an automated reporting service that simplifies the daily task of gathering balance information. It enables the user to procure details on movements in the various accounts being maintained with the bank. This gives a tighter control and increased transparency on the banks account balances and transactions.

More About Balance Reporting

To support balance reporting, GCP CashWeb interfaces directly with the back office of the bank for System Bank accounts and supports compliance with the SWIFT or BIC for other than system bank Accounts.

Balance Reporting consists of two types:

•    Summary: These reports are broken down by bank, and present a summary of account balances.

•    Activity: These reports give transaction wise activities for a given account for the selected period or day.

GCP CashWeb Balance Reporting will display the account balances and account activity for the current date and previous dates.

□         In case of System Bank Accounts: Balance reports for the ‘Same Day’ provide information on the balances of different accounts with the system bank, for the current day and would be fetched from the Host on real-time basis. For prior day balances, previous day’s balances are displayed for a date range depending upon user selection. For Prior day balances GCP CashWeb refers to data as received from designated interface at the end of the day (EOD).

□         In case of other bank Accounts: Balance reports for ‘Same Day’ provides information on the balances of different accounts with Non-System Bank , and would be sent to system bank by the designated interface. For ‘Prior Day’ balances, the previous day balances are displayed for a date range depending upon user selection and is received by GCP CashWeb from designated interface at EOD using standard SWIFT or BIC statements.

The GCP CashWeb Balance Reporting solution comprises of :

•    Client User functionality through GCP CashWeb using which the client can setup masters for balance reporting, activity reporting and querying.

•    Bank User functionality through backend and GCP CashWeb using which the bank can set up the master for the client to undertake balance reporting in GCP CashWeb Client console.
