About Master Copy

When a new corporation or client is added to GCP CashWeb, you need to set up all the masters for that corporation or client. In order to expedite the master setup definitions, GCP CashWeb provides the Master Copy feature that enables you to copy the client-related masters from existing client setup to the new client setup of the same or different Corporation and is applicable only for Enabled and authorized records.. The Master Copy feature will only be applicable for the initial master setup. You can further edit the master details that you have copied, if required. However, any subsequent changes done to the original set-up cannot be copied to other clients.

This feature greatly reduces the manual efforts and time involved in setting up the masters for each of your new client

The Master Copy feature is segregated into four categories as follows:

□         Inter Corporation: This category allows you to copy only the Category master.

□         Inter Client: This category allows you to copy the Client Mask, Client Parameters and MyProduct masters.

□         Intra Corporation: This category allows you to copy the User MyProduct, User Account and User Client masters.

□         Intra Client: This category allows you to copy the MyProduct Account, User MyProduct, User Account and Authentication Matrix masters.

Defining a Master Copy:

Before creating a Master Copy, you need to define the master setup for the existing corporation or client from which the masters have to be copied. You must also set up the new corporation and the new client in their respective masters. Once the new corporation or client is defined and authorized in GCP CashWeb, you can invoke the Master Copy feature. This feature can be used for the initial setting up of masters for the new corporation or client. Any subsequent changes made to the original setup cannot be copied to the new corporation or client. Once the masters are copied to the new corporation or client, you need to authorize these copied masters manually.

Note: The details of the 'Master Copy Entry' screen differ depending on the option selected for the Copy Type field which displays four categories: Inter Corporation, Inter Client , Intra Corporation and Intra Client. This depends on whether you have logged in as Client or Admin.  The subsequent topics explain each category in detail.

As  Bank Admin login:

•    For Copy Type: Inter Corporation

This category enables you to copy masters from one corporation to a different corporation. In this case, you can copy only the Category master.

□         Master To Copy: This field displays the Category master that you can copy from source corporation to target corporation

□         Note: If the Category master already exists in the Target Corporation, the master copy feature is not applicable since GCP CashWeb does not override the details.

Source

□         Corporation: Click the Seek List icon. The “Corporation Help” list appears displaying all authorized corporation definitions set up in the Corporations master. Select the corporation from which you want to copy the Category master.

Target

□         Corporation: This field is auto populated with the Corporation of the logged in Client user.

□         CopyRef: Type a unique reference number to identify the master copy.

•    For Copy Type: Inter Client

This category enables you to copy masters from one client under a corporation to a different client under same or different corporation. In this case, you can copy the Client Mask, Client Parameters, Authorization Matrix and MyProduct masters.

Note:

The Master Copy feature is not applicable in the following cases:

-         If My Products have already been defined in the Target Client

-         If transactions already exist for the Target Client.

-         If the Source Clients’ bank product is not defined for the Target Client.

□         Master To Copy: Select the master that you want copy from source client to target client. The available options are Beneficiary Master, Ordering Party and My Product.

Source

Target

□         Client: This field is auto populated with the logged in Client user.

□         CopyRef: Type a unique reference number to identify the master copy.

•    For Copy Type: Intra Client

This category enables you to copy masters from one product/user/account to a different product/user/account under the same corporation and client. In this case, you can copy the MyProduct Account, User My Product, User Account and Authentication Matrix masters.

□         Master To Copy: Select the master that you want copy from source client to target client. The present available option is Authentication Matrix.

Note:

You can copy the accounts from source user to the target user only if the accounts are assigned to the source user.

The Master Copy feature is applicable in the following cases:

-         If source product and target product are payment products.

-         If the Authentication Level for source product and target product is same, either Product or Account.

-         If the Authorization Level is Account, the Authentication Matrix should not exist in target MyProduct and target Account.

-         There should not be any outstanding AVM for the target.
