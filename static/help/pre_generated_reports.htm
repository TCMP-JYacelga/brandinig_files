Pre Generated Reports

To view Pre Generated reports:

1.      Specify the From/ To Date and click Filter to view all the reports falling within that parameter. The information on that report generation is displayed under the following heads:

•        Media

•        Run On

•        Status

•        Copies

•        Pages

•        Remarks

•        File Name

The Media column displays the options provided for generating the reports.

  • Print: Print via Printer

  • Pdf: Open with Acrobat Reader

  • File: Download contents to a file

  • Email: Send the output over e-mail

  • Screen: Output preview on screen

 

2.      Click the option under Media - E.g. Click Print (in case you are connected to a printer) to generate a printed output of the Report.

OR

Click Back to go to the previous screen
