Instrument Status Query

Collection >> Collection Query >> Instrument Query

The Instrument Status Query retrieves the status of various instruments within a deposit. It retrieves data on the status of instruments as they pass through various stages in the transaction workflow. You can also view the liquidation type against each instrument that is listed.

Viewing the Instrument Status:

You can filter by Date or click Advanced Filter to retrieve the required instruments.

•    Specify the Advanced Filter criteria:

□         Inst. No.: Type the number to view the instruments with that number.

□         Inst. Amount: Select <, >, or = as the filter criterion. Type the amount for the selected filter criterion. Data is filtered on the basis of the amount criteria specified here.

□         Inst. Status: Select the instrument status. The options available for selection are:

Lost

Returned

Paid

Realisation Details Awaited

□         Deposit Slip No.: Type the deposit slip number to retrieve deposit slips with this number.

□         Deposit Date From and Deposit Date To: Type the start and the end dates of the period to specify the range of deposit dates. The query will retrieve deposits with the dates falling within the specified period

Note:

-         If you do not specify the Deposit Date From, the system retrieves all records that are available in the database till the specified date.

-         If you do not specify Deposit Date To, then by default, the system takes the application date. It retrieves all records from the specified date till the application date.

-         If you do not specify the From and To dates, the system retrieves all records that are available in the database till the application date.

-         The Deposit Date To has to be greater than or equal to the Deposit Date From.

E.g.: If you type the Deposit Date From as 01/05/2010 and the Deposit Date To as 01/06/2010, then the query retrieves deposits with the deposit dates falling within the specified period.

□         Credit Date From and Credit Date To: Type the start and the end dates of the period to specify the range of instrument credit dates. The query will retrieve instruments with the credit dates falling within the specified period.

Note:  

-         If you do not specify the Credit Date From, the system retrieves all records that are available in the database till the specified date.

-         If you do not specify Credit Date To, then by default, the system takes the application date. It retrieves all records from the specified date till the application date.

-         If you do not specify the From and To dates, the system retrieves all records that are available in the database till the application date.

-         The Credit Date To has to be greater than or equal to the Credit Date From.

E.g.: If you type the Credit Date From as 01/05/2010 and the Credit Date To as 01/06/2010, then the query will retrieve instruments with the entry dates falling within the specified period.

□         Product: Select the product for which you want to view the instrument details. The values that are displayed in the list are available from the Products master.

□         Arrangement:  Select the client product arrangement for which you want to view data. The values displayed in the list are available from the Client Product Arrangement master.

□         Pickup Location: Select the pickup location of the deposits. The values displayed in the list are available from the Clearing Locations master.

□         Pickup Point: Select the pickup point from where the deposits were picked up. The values displayed in the list are available from the Client Pickup Point master.

•    Click Continue. The grid screen appears displaying a list of records satisfying the specified filter criteria.

•    OR

•    Click Cancel to close the screen.

The Instrument Status Query screen displays all the available instruments, the liquidation status, and the total amount of these instruments.

1.      To view details of any instrument, click View. The Instrument Details screen opens where all the information related to that instrument like deposit details, liquidation details and advice info can be viewed.

□         Product: View the product that was selected for the instrument

□         Arrangement: View the client product arrangement as defined in the Client Product Arrangement master.

□         Pickup Location: View the pickup location of the deposits as defined in the Client Pickup Locations master.

□         Pickup Point: Select the pickup point from where the deposits were picked up. The values displayed in the list are available from the Client Pickup Point master.

□         Payer: View the name of the Payer.

□         Deposit Slip No.: View the deposit slip number

□         Deposit Date: The date on which the deposit was initiated.

□         Credit Date: The date on which the deposit was credited.

□         Credit Ext. Date: The date on which the credit has been extended to the client after return liquidation.

□         Line Rev. Date: The date on which the Credit Line is reversed.

□         Central Bank Reference: Displays the Central bank reference number.

□         Deposit No.: Displays deposit number.

□         UDE Deposit No.:Displays UDE Deposit number.

□         Instrument Ref. No.:Displays Instrument reference number.

□         Central Bank Reference : Displays Central Bank Reference number against which the transaction is cleared.

□         Purpose Code: Displays the purpose code of the transaction if associated with any.

□         View the Liquidation Details in case the instrument has got liquidated as well as the Advice Info. Details of that Instrument in the below pane.

2.      Click Back to return to the 'Instrument Status Query' screen.

3.      To download the transaction details, click Download- you have the option to 'Open' or 'Save' the Instrument Status Query details in .xls format.
