Deposit Status Query

Collection >> Collection Query >> Deposit Query

The Deposit Status Query retrieves the liquidation status of various deposits that are entered in GCP CashWeb. On specifying the filter criteria, the grid screen displays the summary information of various deposits.

Viewing the Deposit Status:

You can filter by Date or click Advanced Filter to retrieve the required deposits.

•    Specify the Advanced Filter criteria:

□         Deposit Slip No.: Type the deposit slip number to retrieve the deposit slip with this number. You can retrieve the exact record by specifying this value.

□         Deposit Amount: Select <, >, or = as the filter criterion. Type the amount for the selected filter criterion. Data is filtered on the basis of the amount criteria specified here.

□         Status: Select the deposit status. All deposits, the transaction processing status of which is similar to the one selected here are retrieved. The following options are available:

All

Open

Closed

Rejected

Being Processed

□         Credit From Date and Credit To Date: Type the start and the end dates of the period to specify the range of credit dates. The query will retrieve deposits with the credit dates falling within the specified period.

Note:

-         If you do not specify the Credit From Date, the system retrieves all records that are available in the database till the specified date.

-         If you do not specify Credit To Date, then by default, the system takes the application date. It retrieves all records from the specified date till the application date.

-         If you do not specify the From and To dates, the system retrieves all records that are available in the database till the application date.

-         The Credit To Date has to be greater than or equal to the Credit From Date.

E.g.: If you type the Credit From Date as 01/05/2010 and the Credit To Date as 01/06/2010, then the query retrieves deposits with the credit dates falling within the specified period.

□         Product Description: Select the product for which you want to view the deposit details. The values displayed in the list are available from the Products master.

□         Arrangement: Select the dispatch bank product arrangement to retrieve records on the basis of this combination. The values displayed in the list are available from the Dispatch Bank Product Arrangement master.

□         Pickup Location: Select the pickup location of the deposits. The values displayed in the list are available from the Clearing Locations master.

□         Pickup Point: Select the pickup point from where the deposits were picked up. The values displayed in the list are available from the Client Pickup Point master.

□         Pickup From Date and Pickup To Date: Type the start and the end dates of the period to specify the range of pickup dates. The query will retrieve deposits with the pickup dates falling within the specified period.

□         Deposit No: Enter the deposit number.

□         UDE Deposit no.: Enter the UDE deposit number if known.

Note:

-          If you do not specify the Pickup From Date, the system retrieves all records that are available in the database till the specified date.

-         If you do not specify Pickup To Date, then by default, the system takes the application date. It retrieves all records from the specified date till the application date.

-         If you do not specify the From and To dates, the system retrieves all records that are available in the database till the application date.

-         The Pickup To Date has to be greater than or equal to the Pickup From Date.

E.g.: If you type the Pickup From Date as 01/05/2010 and the Pickup To Date as 01/06/2010, then the query retrieves deposits with the pickup dates falling within the specified period.

Click Continue. The grid screen appears displaying a list of records satisfying the specified filter criteria.

OR

Click Cancel to close the screen.

The Deposit Status Query screen displays all the deposits along with their relevant information like liquidation status, the total amount of these deposits, etc.

1.      Click View to open the Deposit Details screen where all the information related to that deposit is listed under Deposit Summary and Summary Detail.

2.      Click the Detail button to view further details of the deposit.

3.      Click View to view the instrument, liquidation and advice info details of each instrument within the deposit. Click Back till you come to the 'Deposit Status Query' screen..

4.      To download the transaction details, click Download- you have the option to 'Open' or 'Save' the Deposit Status Query details in .xls format.
