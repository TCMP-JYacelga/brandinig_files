Currency Based Authorization

Authorization matrix can be set up in different currencies. A seek list allows you to select the Currency at AVM & SVM for authorization types -  Product & Account. It is applicable for both Payment & Collection MyProducts.

Currency Code is prefixed to the “Limit From” & “Limit To” of the Detail Records of the AVM/SVM.

Processing

1.      Currency selected for the first slab is applicable to all the slabs set thereafter. Any modifications required to be done should be done in the initial slab and will be defaulted to the remaining slabs.

Example:

Select currency USD and Enter slab 0-100.

Then the Currency field is defaulted to USD while setting up the next slab of 101 to 1000. You cannot change the currency here.

You need to go to initial level i.e.0-100 to edit the currency.

 

2.      If the transaction currency and authorization matrix currency are different, system applies conversion of transaction currency to authorization matrix currency for verification of slabs.

3.      System validates whether currency pair is maintained either for Product Currency- AVM/SVM Currency or for Debit Account Currency - AVM/SVM Currency. This validation is an additional validation to the existing validations which are applicable for Authorization Matrix.

4.      In order to achieve this, user has to operationally ensure to maintain currency pair either in Product Currency- AVM/SVM Currency or in Debit Account Currency -AVM/SVM Currency.

Example

Setup

UDE Bank Product = SGPBANKCHQ

(Bank products under this UDE: SGPDDAUD, SGPDDGBP, SGPDDEUR, SGPDDUSD, SGPDDINR)

Transaction initiated for DD issuance of USD 100 & Product derived by system SGPDDUSD & AVM is maintained in Currency- AUD (Australian Dollars)

Seller currency = HKD

Product Currency of the product SGPDDUSD = USD

Debit Account Currency for the product SGPDDUSD = SGD

Authorization Matrix Currency for MyProduct DDFINDEPT = AUD

As per existing setup, Bank maintains Currency pair in USD-SGD or pairs in Triangulation Currency (USD-HKD & SGD-HKD) in order to issue Demand draft  in USD  by debiting the SGD account.

In addition to the above, Bank has to maintain Currency Pairs in USD-AUD (Product Currency vs. AVM Currency) or SGD-USD (Debit Account Currency vs. AVM Currency), in order to achieve Authorization Matrix validation.
