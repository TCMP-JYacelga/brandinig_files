About Account Activity

The Account Activity screen displays the account activity records i.e. transaction details credited/debited on the account, in the sequence as received from the intra-day account activity interface. This report displays your account movement for the current day or for a given period with process details such as Posting and Value date and the total Credit or Debit amount for the period.

For current date account activity details, system will interface with host system and retrieve the records for current date. Whereas, for previous date system will refer the records uploaded via EOD Account Balance Upload.

In the Account Balance screen, click the Account Number hyper link to open the Account Activity screen. .

§        You can select an account and  Filter the search to view the Account Activity of the selected period.

□         Select Account: Select the account/ IBAN number from the list displaying all the accounts linked to the user.

□         Posting Date: Select the Date for which Account Activity is to be displayed. Default value is 'Today'. Other Options are (Yesterday, Current Week, Last Week, Current Month,Last Month,Date Range). Select Date Range and specify the range to view the account activity of a particular period.

Click Refresh to view to view the relevant account activity.

 

§        View the Opening and Closing Available Balance as well as the Opening and Closing Ledger Balance on the selected posting date.

Note:

The Opening / Closing Available and Ledger balances displayed here are the last recorded balances if any of the from and to dates fall on holidays or weekends.

Eg.

Say a User enters

From Date: as 04/03/2012 (Sunday)

To Date: 17/03/2012 (Saturday)

Then the system fetches closing available balance and closing ledger balance as of 03/03/2012 and are displayed against the opening available balance and opening ledger balance fields respectively.

The system fetches the closing balance as of 15/03/2012 and are displayed against the closing available balance and closing ledger balance fields respectively.

 

The detailed Account Activity is further displayed under the following heads:

§        Posting Date: This column indicates the date on which the transaction was hosted on to the application.

§        Value Date: Date on which the amount is credited or debited as scheduled.

§        Transaction Narration: This column holds a brief description indicating the nature of the transaction.

§        Amount: This column displays the transaction amount.

§        Cr/Dr Indicator: This column indicated the type of transaction (Credit or Debit)

□         Click Enrichments of a selected account to view the enrichments details as defined in the Client A/C Enrichments master.

□         Click Additional Info icon of a selected account to view the more details of that transaction like the transaction Code and description as well as the BAI transaction code.

Other Actions that can be taken:

1.      Click XLS icon  to download screen data in XLS format.

2.      Click PDF icon to download data in PDF format.

3.      Click the Back button to go to the Account Balance page.
