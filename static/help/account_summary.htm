Account >> Account Summary

Bank Transactive’s Account Management module offers an automated balance reporting service that simplifies the daily task of gathering balance information. It enables you to procure details on movements in the various accounts being maintained with the bank. This gives you tighter control and increased transparency on the bank account balances. To support balance reporting, GCP CashWeb interfaces directly with the back office of the bank for System Bank's Accounts.

  • You can access this Account inquiry feature in  GCP CashWeb only if you are assigned rights to do so in GCP CashWeb Admin Console.

  • Ensure that all the accounts for which you want to generate the report are registered with the system through the Account master and Account Mask master.

  • GCP CashWeb also provides the facility to generate the reports for the accounts held with other banks. For this, you must ensure that these accounts are registered with the system and the account information is uploaded in the system using the MT940 message using SWIFT.

  • You can avail of different types of account management reports that are generated by the Account Management module of GCP CashWeb. These options enable you to view account balances for the current day, the prior day, for time periods in history or float balances. You can also view the Account Movement and account statement reports.

  • The information displayed in the Account Management module is populated based on the data received from Host.

About Account Summary

The Account Summary screen offers top-level reports that are the points of origin for the transaction level reports. This report is broken down by bank as per client specification, and provides you a summary of your various account balances across banks. This report displays data for the current balances of the system bank accounts fetched on a real time basis as received from the Intra-day Account Balance interface that is triggered when the user clicks on the ‘Account Summary’ menu.

  • To view Summary Reports:

  • Complete the criteria as specified in the Account Summary.

    Specify the following criteria:

    • Quick: This radio button is selected by default, and within a pre-defined waiting time, if balances are not received, system will display the last available balance.

    • Current: On selection of this radio button, system after a pre-defined waiting time will display the current balances, else will display the last available balance.

    Note: Either Quick or Current radio button can be selected at a time.

    • Equivalent Currency: This field will display all the Currencies listed in the Currency Master as equivalent currencies.

    • Show Favorite: This button is selected as default and will show only those records marked as favorite (Yellow or Green). For System Bank Accounts, based on user-account linkage, system will read the balance for the Favorite accounts. If data is not retrieved within the specified time period, then system will display the last available balance.

    • Show All: On selection of this radio button all the records will be displayed. Based on user-account linkage system will read the balance (For all system Bank Accounts). If data is not retrieved within the specified time period then system will display the last available balance.

    Note: Select either Show Favorite or Show All at a time.

    Click the Refresh button the search based on the above criteria.

    The tabs applicable for data retrieval are as follows:

    • Facility: Retrieve all valid accounts of the corporation (with User Account linkage) that are grouped under Facilities available in the system like Current, Term Loan, etc.

    • Currency: Retrieve all valid accounts of the corporation (with User Account linkage) that are grouped based on Account Currency.  

    • Bank: Retrieve all valid accounts of the corporation (with User Account linkage) that are grouped based on the Account Bank.  

    • Group: Retrieve all valid Accounts of corporation (with User Account linkage) that are grouped based on user defined Account Groups. If an account belongs to multiple group it will be displayed multiple times (Once in each group it belongs).

    E.g. If account belongs to Finance group (Parent group Maharashtra, India ) it will be displayed in Finance Group. It same account linked to HR Group it will also be displayed in HR Group.

    All the accounts which do not belong to any group will be displayed under- 'Not Grouped'

    The grid details are as follows

    • Favorite: Click this button in case you want to save the record under favorites.

      This is a toggle button with 3 states.

      • Grey: Non-Favorite

      • Green: Favorite but not saved (Will be green for current session only)

      • Orange: Saved favorite records).

      If status is Grey (Default), on click it changes to Green (Selected for Favorite). If user clicks Save Preferences, all the records in Green will be saved as favorites and will become Orange at next login. If clicked again, it changes back to Grey (Non-Favorite) .

    • Company Name: Displays the name of the client.

    • Bank Name: Displays bank in which the account is held.

    • Account Number: Displays the Account/IBAN number.

    • Account Name: View the Account Description as maintained in Account Name Master.

    • Account Country: Displays the country of the Account

    • Current Balance: View the Current Balance of the Account with Account Currency Sign

    • Ledger Balance: View the Ledger Balance of the Account with Account Currency Sign.

    • Bank Name: Displays bank in which the account is held.

    • Sub Total: View the sub total of Current Balance and Ledger Balance in the Equivalent Currency selected above with Currency Sign.

    • Grand Total: View the total of the Current and Ledger Balance minus all deductions.

  • Click Save Preferencesto save the current preferences. On next login, records can be viewed based on current saved preferences (Favorites setting, Quick/ Current flag setting, etc).

  • Click XLSto save the current screen data in xls format with Account Data information fields.

  • To view Account Activity, click on the Account Number hyperlink

  • To view Account Balance, click the History icon  hyperlink
