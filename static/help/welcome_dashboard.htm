About the GCP CashWeb module.

Depending upon the set up done by the bank, all the screen labels of the entire CashWeb can be displayed in the user defined language. The GCP CashWeb solution at proposed bank will comprise of the following:

•    Client User functionality through  GCP CashWeb – using which the clients can originate cash management transactions and view the transaction status and reports.

•    Bank User functionality through - GCP CashWeb – using which the Bank Administrator can initiate and process the transactions on behalf of the customers as rand when required.

The use of the above functionality is supported through ancillary modules such as Charging, Accounting, Reporting and Querying, etc. The Cash Management system is interfaced to these ancillary systems within the bank and also with the customers.

The 'Welcome' Dashboard greets you as soon as you log in. This screen allows you to get a broad overview of your account status on a single screen in an easy to understand and graphical manner.

□         Click here to view the Welcome Dashboard image.

The Dashboard displays the following widgets:

□         Awaiting User Actions: This widget shows the number of transactions awaiting necessary action.

□         Account Balances: This widget is available only if the system parameters are set accordingly. This widget displays your account balance in graphical format. The drop down list displays the accounts that are selected under User Preferences available on the menu bar.  The data relevant to the selected accounts will be displayed here.

User Preferences is available only if the Account Balance widget is visible.

•    To select the preferred accounts:

□         Click to open the User Preferences page.

□         Under Dashboard Accounts, select a maximum of three accounts whose balances can be viewed in a graphical format on the dashboard.

Ref: For more details refer to Conventions- User Preferences.

□         Announcements and News Feed: This widget displays any special announcements or news feeds from the bank, that you can view or download in .pdf format. Mouse roll over on each announcement will display additional information of the same.

□         Messages: All the Event Notification related messages as well as any e-mails sent by bank can be viewed in under this widget. It holds a maximum of 5 messages with unread messages displayed in bold.

□         Reports: This widget displays last five reports that you have generated across all modules. To view all the reports, click the Reports (Last Five) hyper link. This takes you to the View Reports screen that appears under the Reports menu. You can view the reports based on the rights assigned to you.

The menu bar displays the modules available to the logged in customer. They are:

□        Admin

□        Payments

□        Account

□         Reports

The left menu column displays the following:

□         Quick Links: This box holds the most frequently used menus that you have stored for easy access using the “Add to Quick Links” button.

Ref: For more details on Quick Links, refer to Conventions- Add Current Page to Quick Links & Remove Current Page from Quick Links.

□         Exchange Rates: This widget displays the 'Buy' and 'Sell' rates of various currencies as specified by the bank, against the Seller Base Currency of the Client. These currencies are displayed in alphabetical order to all clients maintained in the system. Currency rates are rounded to 4 decimal values (999.9999).
