About Customer Upload and Download

The Customer Upload Download feature enables you to upload or download data addressed to the bank. This feature supports all file formats, thus enabling you to upload any document or image.

An uploaded file needs to be authorized before it can be transmitted to the bank. An uploaded file resides in a temporary server location, from where it is transferred to the permanent server location after authorization. If the file is to be deleted, it should be done prior to authorization.

You can also download the transaction details uploaded by the bank admin on your behalf provided the file is authorized. GCP CashWeb does not perform any validations during downloading a file.

The Customer Upload Download screen displays all the files that are uploaded in GCP CashWeb along with their current status.

To upload a file:

1.      Click the Upload button to open the Upload File pop-up screen.

2.      Specify the following details to upload the file:

§        File: Click the Browse button and navigate to the path where the file to be uploaded is placed on your local machine. If you are entering the name of the file, ensure that you are specifying the absolute path of the file.

§        Description: Type a brief description of the file.

3.      Click Upload to upload the file. You view the message indicating whether the file has been uploaded successfully or not in the header area. Click Cancel to cancel the task.

Note: If you have uploaded files that you do not want to enable for further use in the system and if those users are not yet authorized, you can Discard them.

An uploaded file needs to be authorized before it can be transmitted to the bank and will not be valid unless it has been authorized based on the process of the Maker-Checker. The checker can view the files in this screen and authorize the same provided the rights to authorize are assigned to that checker.

You can filter the search by 'day' or use the Advanced Filter where you can specify the File Name, and the From\To Date retrieve a particular account for authorization.

To authorize an uploaded file:

§        Select the file(s) that you want to authorize. The Authorize icon gets activated. Click the Authorize icon to accept an uploaded file.

OR

§        Select the Reject check box to reject a file. While rejecting a file you will be prompted to fill in the reject reason in the pop up screen.

The header area displays the message indicating whether the file has been authorized or rejected successfully .

To Download a file

Click the Downloadicon to download a selected file. System provides you an option to Open, Save or Cancel the download.
