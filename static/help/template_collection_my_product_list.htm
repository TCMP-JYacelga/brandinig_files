Template Collection My Product List

Admin >> Template >> Temp Coll MyProd

The Template Collection My Product master allows you to define a My Product template that can be attached to a client. This reduces the task of setting up masters for each Clients- MyProduct combination. This master is a replica of Collection My Product master.

Topics Covered

•    Defining a Template Collection My Product

•    Editing a Template Collection My Product

•    Viewing a Template Collection My Product

Defining a Collection My Product Template:

Note: To retrieve an existing template, define the filter criteria. You then have the option to edit the same.

1.      Click Add New MyProduct to go to the Template Collection My Product Master Entry screen.

Note: Template Collection My Product fields are similar to the Collection My Product master.  For field level details, refer to the topic Collection My Product Master Entry.

2.      Click Save to save the template and return to the Template Collection My Product List screen where you can view the newly defined template with status as New. This implies that it is a new addition and yet to be authorized. You have the option to edit or discard the newly defined template.

3.      Click Back if you want to go to the previous page without defining the template.

To enable the My Product template, it is mandatory to authorize it based on the Maker-Checker concept. The checker can view the new definition in the Template Collection My Product Authorization List screen and authorize the same provided the rights to authorize are assigned to that checker.

A successfully authorized My Product template definition request can be viewed in the Template Collection My Product List screen with its status as Authorized.

Note: If you have created My Product templates that you do not want to enable for further use in the system and if those My Products are not yet authorized, you can Discard them.

 

Editing a Collection My Product Template :

1.      Click Edit to go to the Template Collection My Product Master Entry screen where you can edit a previously defined master.

2.      Click Update to save the changes.

Note:

 You can modify the definitions if they are in any of the following states:

-         New definition: Any new definition is not yet authorized and is in the new state.

-         Modified definition: Any definition that you have modified and not yet authorized and is in the Modified state.

-         Enabled definition: The definition that is authorized and is in the Enabled state.

The edit option is not available for the definitions that have the state as Disabled, Disable Req, Enable Req, Dis. Req. Reject, and Ena. Req. Reject, Submitted, Authorized.

On saving the modifications, the definition state changes to Modified. To enable the definition with the Modified state, the definition has to be authorized.

Viewing a Collection My Product Template:

1.      Click View to go to the Template Collection My Product Master View screen where you can view a previously defined master. You cannot modify the definition in this screen.

2.      Click Back to return to the Template Collection My Product List screen.
