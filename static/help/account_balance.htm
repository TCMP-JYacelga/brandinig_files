About Account Balance

The Account Balance screen offers top-level reports that are the points of origin for the transaction level reports providing accurate and timely information of your cash balance. These reports are broken down by the bank as per client specifications, and enable you to procure details on the movements in the various accounts maintained with the bank. These reports display data for the current balances of the system bank accounts fetched on a real time basis, as received from the Intra-day Account Balance interface, that is triggered when the user clicks on the ‘Account Summary’ menu.

In the ‘Account Balance’ screen view the last updated report of the account. You can also view account balances for the prior day, for time periods in history, if available.

§        You can Filter the search to view the Account Balance based on the criteria selected.

□         Current: This radio button is selected by  default and displays the current balances, On selection of this radio button, system after a pre-defined waiting time will display the current balances, else will display the last available balance.

□         Update: This radio button is selected by default, and within a pre-defined waiting time, in case balances are not received, system will display the last available balance.

Note: You may select either Current  or Update radio button at a time

□         Equivalent Currency: This drop down will display all the Currencies listed in the Currency Master. Select the currency in which you want to view the account summary.

□         Show Favorite: This button is selected as default and will show only those records marked as favorite (Yellow or Green). For System Bank Accounts, based on user-account linkage, system will read the balance for the Favorite accounts. If data is not retrieved within the specified time period, then system will display the last available balance.

□         Show All: On selection of this radio button all the records will be displayed. Based on user-account linkage system will read the balance (For all system Bank Accounts). If data is not retrieved within the specified time period then system will display the last available balance.

On selecting the required filter criteria, click  to view the relevant account balance summary.

Note: You may select either Show Favorite or Show All at a time.

You can also view your account details based on the below criteria. The tabs applicable for data retrieval are as follows:

□         Click Account Type to retrieve all valid accounts of the corporation (with User Account linkage) that are grouped under the different types of accounts available in the system like Savings, Current, Term Loan, etc.

□         Click Currency to retrieve all valid accounts of the corporation (with User Account linkage) that are grouped based on Account Currency.  

□         Click Bank to retrieve all valid accounts of the corporation (with User Account linkage) that are grouped based on the Account Bank.  

□         Click Group to retrieve all valid Accounts of corporation (with User Account linkage) that are grouped based on user defined Account Groups. If an account belongs to multiple groups, it will be displayed multiple times (Once in each group it belongs)

View the status details of the accounts along with the Available and Ledger Balance and the Grand Total based on the criteria selected above.

Note:

-         The Intra-day Account Balance interface is triggered when the user clicks on the ‘Account Summary’ menu or ‘Account’ menu. GCP CashWeb application will receive the response in Intra-day Account Balance interface as follows:

-         Success – The success response message will be received if balances for all the BTR (Balance Transaction Reporting) accounts assigned to the user are available in the response message of intra-day account balance interface. There will be no change in displaying the account balances for this response messages.

-         Partial Success – The Partial success response message will be received if balances for few BTR accounts and negative response for other account assigned to the user are available in the response message of intra-day account balance interface. In this case system will display message as ‘Balance Not Available’ against the BTR account number in ‘Available Balance’ and ‘Ledger Balance’ field for which negative response is received with the other field value as maintained  in the system. This will further allow the user to view the account balance history available in system by clicking the view button against the account.

-         Fail – The fail response message will be received if the negative response message is received for all the BTR accounts assigned to the user in intra-day account balance interface. In this case system will display message as ‘Balance Not Available’ against the all BTR account number in ‘Available Balance’ and ‘Ledger Balance’ field with the other field value as maintained in the system. This will further allow the user to view the account balance history available in system by clicking the view button against the account

□         Bank Name: Displays bank in which the account is held.

□         Sub Total: View the sub total of Available Balance and Ledger Balance in the Equivalent Currency selected above with Currency Sign.

□         Grand Total: View the grand total of Available Balance and Ledger Balance in the Equivalent Currency selected above with Currency Sign.
