Deposit

The Deposit screen allows you to maintain a summary level record of a deposit with multiple instructions.

Entering Deposit Details

The header displays the My Product selected and the Internal reference Number.

1.      Enter the Deposit details.

Note: The fields may vary depending on the MyProduct selected.

□         Collection Currency: Mandatory. Select the required transaction currency from the drop-down list. If collection product is selected, the same will be defaulted to product currency and will reflect in the Instrument entry screen.

□         Collection Product: Mandatory. This field may appear either at Deposit level or at Instrument level depending on the My Product setup. Select the collection product for the specified MyProduct transaction from the drop-down list displaying all the valid collection products.

□         Account Number: Mandatory. Select the Account Number from the list of Client Credit Accounts.

□         Account Currency: This will populate the currency of above selected credit account.

□         Activation Date: This is the date when processing of the collection instruction begins. The activation date has to be a date same as or subsequent to the application date. Defaulted to current application date.

□         Reference: Mandatory. Enter the actual deposit slip number. This slip number helps you while querying and retrieving the required Deposit.

□         Deposit Remarks: Type remarks that are specific to the Deposit entry that you are making. This field can include the purpose for which you are making the Deposit entry.

□         Total Instruments: Enter the total number of instruments you want to include in the deposit .

□         Total Amount: Specify the total deposit amount for the new transaction.

□         Balance Instruments: System automatically calculates and displays the total number of pending instruments to be entered for a deposit.

□         Balance Amount: System automatically calculates and displays the summation of the instrument amounts to be entered for a deposit.

2.      Click the . If the save action is completed successfully, the message – "Save Action Completed Successfully" along with the Deposit Reference Number is displayed and the icon appears.

3.      Click Add Instrument to go to the Instrument entry page where you specify the instrument level details. At this stage the status of the transaction can be viewed  in the Collection center as Draft.

Note: In case the My Product flag is set to Instrument, you have to Submit, Authorize and Send individual instruments at instrument level in the Deposit screen.
