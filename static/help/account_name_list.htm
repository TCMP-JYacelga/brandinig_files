Account Name List

Admin >> Account Settings >> Account Names

The Account Name List screen enables you to define preferred names for the accounts that have been previously defined.

Note: The accounts listed in the Account Name List screen are the accounts that are used for Balance Reporting.

You can filter the search by Account Number and Account Usage to retrieve a particular account.

To define a Preferred Account Name:

§        Select the account number for which you want to modify the account description and click the Editicon.

Note: Only ‘Enabled’ accounts can be edited or given a preferred name. Disabled accounts should first be enabled before you can edit them.

§        The Account Name Edit screen opens where you can define the preferred name to it.

§        The newly defined Preferred Account can be viewed in this screen with its status as Modified.

§        Any action taken on an account has to be authorized in the Account Name Auth List.

 

Viewing the Preferred Account Name:

1.      Click View to go to the Account Name Edit screen where you can view a previously defined master. You cannot modify the definition in this screen.

2.      Click Back to return to the Account Name List screen.
