User Subscription Details

The User Subscription Details screen displays the details of the event notification, the medium through which the notification is to be sent to the user and the condition of the notification. This will enable the client/bank user to identify the details of subscription before assignment.

The User Subscription Details  are listed under:

Notification Details:

§        Subscription and Description: View the Subscription code and the  description relevant to that subscription

§        Subscription Recipient: This indicates the code of the recipient

§        Effective From-To: View the From and To dates of when the Subscription will remain in effect

Notification Mediums:

This indicates the mediums through which the recipient will be notified.

Notification Conditions:

§        View the condition specified for the notification.

§        Click Back to return to the User Subscription Entry screen.

Note: These subscriptions are only available for viewing and cannot be modified.
