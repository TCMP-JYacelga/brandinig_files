Ordering Party Entry

1.      In the Ordering Party Entry screen enter the personal, bank and contact details of the new Ordering Party under the following heads:

§        Personal Details

□         Ordering Party Code: Type a unique code that will be used to identify and reference the ordering party.

□         Name: Type the name of the ordering party.

□         Mobile Number: Type the ordering party mobile number.

□         E-Mail ID: Type the ordering party e-mail address.

□         Resident Status: Select the residential status of the ordering party. The available options are Resident (default) and Non-Resident.

§        Contact Details

□         Address: Type the address of the ordering party

□         Country: Select the ordering party country from drop-down list showing a list of countries as specified by the bank.

□         State: Select the ordering party State from the drop-down list that displays the filtered records of the states within the country selected above.

□         City: Select the ordering party City from the seek list that displays the filtered records of the City within the State selected above.

□         Post Code: Type the postal code of the mailing destination for the ordering party.

□         Telephone Number: Type the telephone number of the ordering party.

§        Additional Details

□         Document Type: In case of individuals, select the document for personal identification from the drop-down list

□         Document ID: Mandatory if Document Type is selected

2.      Click Save to save the details of the Ordering Party. On saving the entry you are automatically directed back to the Ordering Party List screen where you can view the newly added beneficiary with its status as 'New', indicating that it is awaiting authorization.

3.      Click Back to return to the Ordering Party List screen without entering a new record.

4.      If you are eligible for authorization, the record will be available in the Ordering Party Authorization List screen and you can then Authorize the same.
