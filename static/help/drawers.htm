Payer Entry

1.      The Payer Entry screen allows you to set up the personal and contact details of the new Beneficiary under the following heads:

•    Personal Details

□         Payer Code: Mandatory. Type a unique code that will be used to identify and reference the payer.

□         Name: Mandatory. Type the name of the payer.

□         Mobile Number: Type the payer mobile number. You can enter up to two mobile numbers

□         E-Mail ID: Type the payer e-mail address.

□         Resident Status: Select the residential status of the payer. The available options are Resident (default) and Non-Resident.

Payer Bank Details

□         Bank Type: The options are Registered Bank (checked by default), and Adhoc Bank. If you select Registered Bank, the below four fields are seen.

□         Bank Branch Search: Click the Seek icon to select the Bank Branch code. The seek list also specifies the Low/ high value membership clearing code of the listed banks.

Note:

-         In case of registered bank and depending upon the set up done by the bank, bank and branch name in the seek list will show the names in local language also.

-         In case of Adhoc bank, the user needs to input the Bank and Branch name in English.

□         The Bank and the Branch fields get auto populated and uneditable. These fields are displayed in English as well as local language if maintained in local language by the bank.

□         Low & High Value Clear Code: Displayed for registered bank & enter the Low value and high value clearing codes.

□         Swift Code: Enter the swift code.

□         Bank Address: Displays the address of registered bank and can be input for Adhoc Bank.

□         Bank Country: Select the country from the drop down.

 Payer Account Details: Enter the payer account details.

•    Contact Details

□         Name: The name is auto-populated based on the value entered under Personal Details

□         Payer Address: Type the address of the Payer along with the Country, State, City and the Post Code.

□         Telephone Number and Fax Number:  Enter the Telephone Number and the Fax Number

□        Document Type:  Select the document type from the drop down.

□        Document ID:  Enter the document number for the document type selected above.

2.      Click Save to save the details of the Payer. On saving the entry you are automatically directed back to the Payer List screen where you can view the newly added Payer with its status as 'New', indicating that it is awaiting authorization.

3.      Click Back to return to the Payer List screen without entering a new record.

4.      Payer entry can be set up auto authorization or maker checker authorization from the bank end.

5.      If the payer master entry is set up for auto authorization then the created payer is immediately displayed in the list as authorized.

6.      If the authorization is set up for maker checker then the record is made available in the Payer Authorization List screen and you can then Authorize the same.
