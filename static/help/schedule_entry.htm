Schedule Entry

To define a Schedule Entry,

1.      In the Schedule Entry screen, type the necessary information in the fields provided under the following heads:

§        Report Details

□         Media: Select the medium of delivery from the options provided in the Media Seek list. The options are SMTP (Lotus Notes), Default (Print to default printer) and File (ASCII file output).

□         Output Type: Select the Output type for the report. The available options are: PDF- Acrobat Reader File and XLS-  MS Excel File

□         Email ID/Fax Details: Type the Email ID and the Fax numbers in case the report needs to be sent to the customer .

□         Frequency: Select the frequency at which you want the report to be generated. A report can be generated on daily, weekly or monthly intervals. Accordingly, the options available for selection are: Daily, Weekly, and Monthly.

□         Period: Select the intervals of generation for the specified frequency. The values of this field depend on the selection made in the Frequency field.

E.g. If the frequency selected was ‘daily’, then the Period field would show values 0 to 7, representing the number of days in a week. If you want the report to be generated daily, then you would select 0. If you want the report to be generated every alternate day, you would select 1, and so forth. Similarly, if the frequency selected was ‘weekly’, this field would have values ‘Weekly, Fortnightly, Third Quarter and Last Week’. And if the frequency selected was ‘monthly’, this field would have values 0 to 12.

□         Reference Day: Select a reference day. This field is of significance when you select a report to be generated at weekly or monthly frequencies. In case of a weekly frequency, this field allows you to select the day of the week when you would want the report to be generated, and in case of monthly frequency, you would be provided the options to select the day of the month on which you want the report to be generated.

□         Reference Time: Type the time at which you want the report to be generated.

□         Previous Generation Date: This field displays the application date by default and cannot be edited.

□         Next Generation Date: Select the date when the report will be generated next. The selection boxes allow you to specify a date in the DDMMYYYY format.

□         Start Date: Select the date on which you want the generation of the report to start. The selection boxes allow you to specify a date in the DDMMYYYY format.

□         End Date: Select the date on which you want the generation of the report to end. The selection boxes allow you to specify a date in the DDMMYYYY format.

Note: Fields that are displayed under the heading ‘Parameters’ are report-specific filters for the selected report.

§        Report Parameter Details: The filters will depend on the report selected.

2.      Click Save to add the new schedule to the Schedule Definition List with status as New. The same also appears in the Schedule Definition Authorization screen for authorization

3.      All schedule definitions have to be authorized so that they can be generated.

Tip:  Click Back to go to the previous screen
