About Virtual Accounts

GCP CashWeb provides for a facility to its corporate customers to apply online for a series of virtual account numbers against their real account. This screen provides for an exchanges of information online and prevents the customers from making Paper requests. Based on the online requests, bank issues the Virtual Account numbers to their customers and intimates them offline. Virtual accounts enable customers to have an easy way of paying into Clients Accounts without revealing their real accounts. It also helps clients to receive detailed analysis of Receivables against each of these Virtual accounts.

The ‘Virtual Account Issuance List’ screen displays the existing Virtual Accounts of the client.

Adding Virtual Accounts:

§        Click Add Virtual Accounts to open the ‘Virtual Account Issuance Entry’ screen.

§        Successfully added Virtual Accounts appear in this screen with status as 'New'. You have the option to Edit or Discard the Account.

§        If you are eligible for authorization, the accounts are available in the Virtual Account Issuance Auth List screen and you can then Authorize the same.

§        Successfully authorized Virtual Accounts also appear in this screen with the status - 'Authorized'.

Editing Virtual Accounts

1.      Click Edit to go to the Virtual Account Issuance Entry screen where you can edit a previously accounts

2.      Click Update to save the changes.

Note:

 You can modify the definitions if they are in any of the following states:

-         New definition: Any new definition is not yet authorized and is in the new state.

-         Modified definition: Any definition that you have modified and not yet authorized and is in the Modified state.

-         Enabled definition: The definition that is authorized and is in the Enabled state.

The edit option is not available for the definitions that have the state as Disabled, Disable Req, Enable Req, Dis. Req. Reject, and Ena. Req. Reject, Submitted, Authorized.

On saving the modifications, the definition state changes to Modified. To enable the definition with the Modified state, the definition has to be authorized.

Viewing a Virtual Account Issuance Definition:

1.      Click View to go to the Virtual Account Issuance View screen where you can view a previously defined accounts. You cannot modify the account definition in this screen.

2.      Click Back to return to the Virtual Account Issuance List screen.
