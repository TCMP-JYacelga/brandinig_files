Batch Header

The Batch Header screen allows you to maintain a summary level record of a batch of multiple instructions.

The header displays the My Product selected and the Internal reference Number

1.      Enter the Batch Header Details

The fields will vary depending on the Product selected:

□         Payment Product: Mandatory. Select the payment product for the specified MyProduct transaction. If you select MyProduct, the default payment product is checked.

□         Debit Account:  This drop-down field is displayed if Debit Account is at batch level and shows a list of 'Client Debit Accounts'. If 'Payment Type' selected is 'Credit', field label will change to Credit Account.

□         Activation Date Mandatory. This field is defaulted to current application date. You can select a future date in this field but not a past date. The display of this field depends on the level set in the Template menu of My Product Master Entry.

□         Client Reference: This is a user defined unique reference to help you to query your own transaction. The max length should not be more than 16 characters Mandatory

□         Remarks:  Type remarks that are specific to the Batch entry that you are making. This field can include the purpose for which you are making the Batch entry

□         Instruments: Mandatory Enter the total number of instruments you want to include in the batch

□         Entered Instruments:  Non-editable System automatically calculates and displays the total number of instruments entered for a Batch.

□         Amount: Mandatory. Specify the total Batch amount for the new transaction.

□         Entered Amount: Non-editable. System automatically calculates and displays the summation of the instrument amounts entered within a Batch.

Example explaining Entered Instrument and Entered Amount

  • There are 5 instruments within a Batch of total amount 10000 INR. User has entered 3 instruments of 2000 INR each. In this case, system will display the data as follows:

  • Instruments - 5

  • Entered Instruments - 3

  • Amount - 10000

  • Entered Amount - 6000

  • Now if the user enters one more instrument of amount 1500 INR, then system will calculate and display the Entered Instruments as 4 and Entered Amount as 7500.

2.      Click to save the payment information request. On successful save, system displays the message - 'Payment Information Request Successfully saved'. The Internal Reference Number is also generated by the system.

3.      In case you modify the data in any of the editable fields, click to save the changes.

4.      Click Add Instrument to go to the  Instrument Entry screen to add the instrument level details.

Note: Once the instruments are added, system will validate the total instruments and their amounts with the entered Batch instruments & amounts at header level. In case of any discrepancies, system will display an appropriate error message. This validation is performed based on the Zero Proofing flag set at My Product level.
