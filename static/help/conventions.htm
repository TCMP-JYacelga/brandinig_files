GCP CashWeb                                       

 

Conventions

The list of the icons used in this application are explained below.

§        Drop down: Click to view additional information.

Click again to hide the information.

§        These icons are seen on the right hand side in the main menu bar on every screen. Click the icons to view the definition.

Change Your Preference        

This option is available only if the Account Balance widget is available.

□         Click to open the User Preferences screen. The Available Accounts box lists all the accounts of the user. The dashboard page will allow you to see the balances of a maximum of 3 accounts in graphical format.

□         Select the three accounts and click the 'Right Arrow' icon to move them to the Preferred Accounts box. Click the 'Save Preferences' icon to save the selection. The balances of these three accounts can now be viewed in the Welcome Dashboard screen.

Switch Client

□         The Switch Client feature enables you to work on behalf of another client within the same corporation in order to initiate payments or generate reports etc. This feature allows you to switch to a particular client within the same corporation provided you have the ‘Across Client’ feature enabled as well as User Client relationship privileges. You can switch client without logging off from the application. Once you switch to a particular client, GCP CashWeb displays data relevant only to that client.

□         When you click Switch Client icon, the Switch Client screen appears showing the client name and description. Select the Switch check-box against the client name whose data you want to access. You now have the authority to update the data for the selected client.

Change Language

□         Click the Change Language icon to change the screen language.

Add Current Page to Quick Links

□         Click Add Current Page to Quick Links icon if you wish to add a currently open page that you access more often, to Quick Links. On click, the same will be visible under the Quick Links tab in the left menu bar. Or open to the screen that you want to save for quick access. E.g. Open the 'Payments' center screen and click the Add Current Page to Quick Links button. The same will appear under Quick Links.

Remove Current Page from Quick Links

□         Click Remove Current Page from Quick Links icon to remove a currently open page from Quick Links. Or open to the screen that you want to remove from under Quick Links. E.g. Open the 'Payments' center screen and click the Remove Current Page from Quick Links button. The same will disappear from under Quick Links.

 

□         View the other commonly visible Icons on the screens. 

New icon, opens the pop up window to select MyProduct for a new Payment Initiation. Also used for adding a newly generated instrument.

Submit icon- close the current transaction and "submit for authorization".Submit is only enabled if the transaction status is Draft and is not yet closed.

Authorize icon, used to approve Transaction and Stop pay request

Send icon, will allow you to send authorized transaction to bank

Reject icon, used to reject the transaction pending for My Auth as well as stop payment request. Enter a reject reason, in case of multiple selected transaction, same reason will be applied to all transactions selected.

Hold Payment- Hold a transaction pending for Send.

Release Payment- Release a transaction that was on hold and Send to bank

Stop Payment- Raise a request for stopping the transaction.

Delete payment. In case of Draft, this action will delete the transaction; in case of Close transaction this will mark the transaction as "Delete/Scrap/Soft delete". In case of Instruction definition, it allows you to 'Execute' the instruction.

Downloads the data displayed on screen into an excel file.

History-  Displays the action taken on the transaction. New, Closed, Authorized, Send, Stop, Hold, Stop -Auth etc. along with the User Id for the action taken on the transaction and the date and time of the action in the local seller time-zone.  

Edit icon is enabled only if batch can be edited,i.e. when the batch status is Draft, Repair or Rejected

View Details- Displays the details of the selected definition

Clone icon is enabled only if you have edit rights to the Payment entry. When you click on Clone, the transaction is copied and you are navigated to the batch screen or instrument screen based on whether you are cloning a batch or an instrument. You can view, edit and save the transaction. All the information of the current transaction will be copied to the newly created transaction except for value date (will be set as current date). Internally populated fields like rate will be recomputed on save

Save/Upload a transaction.

Online transaction - displayed next to MyProduct

Transaction tagged as "Confidential" - Displayed next to MyProduct

Uploaded transaction or batch - displayed next to MyProduct
