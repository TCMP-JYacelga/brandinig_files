Payments

The GCP CashWeb Payments module  enables you to efficiently manage your payables, access the system for transactions and reporting or outsource the entire payables management function to the bank for doing transactions or reporting on behalf of the clients.

Click the Payments menu to open the Payments Dashboard that gives you a broad overview of your payables on a single screen in an easy to understand and graphical manner.

Click here to view the Payments Dashboard screen

About the Payment Dashboard

□         Top Five Payment Products: This widget displays the top five Payment products currently in use. This is a system generated list that also shows the transactions done in the Last and Current quarter using these products.

□         Payment Statistics: This widget displays a graphical representation of the transaction volumes across all the Payment products for a currency on a per-day basis.

□         Payment Status: This widget displays the transaction status of all the records initiated by the Customer, the Bank on behalf of customer, and Stop Requests if any, from both.

□         Payment Exceptions (Last Seven Days): This widget displays all the transactions of the last seven days, that have moved to Reject Repair.

□         File Upload Status: This widget displays the status along with the Name and the Date and Time, of the last five files uploaded by the customer.
