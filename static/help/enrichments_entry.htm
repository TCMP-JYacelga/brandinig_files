Enrichments Entry

The Enrichments Entry screen allows you to define new enrichments pertaining to Balance Reporting, to an existing account or to all the accounts of a particular client.

Note: The enrichments defined in this master can be viewed in the Account >> Account Summary >> Account Activity >> View Enrichments

To define a new Enrichment Entry:

1.      Specify the following fields,

§        Enrichment Level: Select Account or Client depending on which level you want to apply the enrichments.

In case you select Account at 'Enrichment Level', select the Account Code from the Account Seek list showing all the accounts of that client.

In case you select Client at 'Enrichment Level', this field is non-editable. The enrichments defined will apply to all the accounts held by that client.

§        Required: In case you select the ‘Required’ check box for a particular enrichment, a transaction request for which the enrichment is specified cannot proceed further unless the enrichment is entered, i.e. the enrichment is mandatory for that transaction.

§        Screen Label: This indicates the enrichment field label as you want it to appear on the screen, e.g. Invoice No

2.      Click Update to save the Enrichments. You can view the newly defined entry in the Enrichments List screen with status as New.

Note: To modify an existing record, click the Edit icon to go to the 'Enrichments Entry' screen where you can make the required changes. Click Update to go back to the Enrichments List screen where you can view the record with its status as ‘Modified’.

To enable the Enrichments, it is mandatory to authorize it based on the Maker-Checker concept. The checker can view the new definition in the Enrichments Authorization screen and authorize the same provided the rights to authorize are assigned to that checker.
