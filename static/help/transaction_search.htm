About Transaction Search

Transaction Search allows you to customize filters on Account Balance and Account Activity screens in the form of Templates. Templates reduce the time and effort to select a set of filters every time you login in the system and search for certain information for a said client. Save a template with the necessary fields specifying the filter conditions. Based on this template, system displays the details on the summary page.

The Templates screen header displays the list of existing templates in the system. In case you do not require a template, you have the option to Delete it.

You have the option to define a new template,add or edit an existing template, or select an existing Template to view its details.

Defining a Template:

1.      Specify the following:

§        Template Name: Type a unique identification code for your Template. If you have selected an existing Template, this field need not be changed.

§         Field: This field allows you to customize the Template as per your requirements. Select a field from the drop down list, enter the relevant values and click .The selected field gets added to the template. You have the option to add all the six fields available in the dropdown. You may also delete a field that you have already added to your Template.

2.      Click   to save a new or modified Template, or save it with a new name ().

Note: You can view or modify a Template provided you have the right to do the same. This is based on the value specified for the "Is Personal" check box. In case the check box is not selected, the template defined will be a global template and will be visible to all the user of the same client, if selected,  then the template will be visible only to the user.

3.      To modify an existing Template, click to clear all the existing details or Delete only those fields that are not required.

4.      Select a template and click Search to search for transactions based on the template or download the output in Xls format.

§        You can also create a New Template by selecting a set of filters to suit your requirement, customize it and name it with a unique identification code.

§        Click Clear to clear existing details of a selected template. To modify the template, click Add Field to add a new field to your template or Delete an existing field. You can also add new fields and/or delete existing fields.

§        Select a template and click Search to search for transactions based on the template or download the output in format.
