GCP CashWeb                                       

 

Download List

Reports >> Download Data >> Download Now

GCP CashWeb provides for a transaction status file download for transactions initiated via the File Import functionality. The status file is also referred to as Reverse file download which can be downloaded as per scheduled frequency or can be On-Demand. This file can be used by customers to update their backend systems for transaction status update. Reverse file is generated at a file level and follows transaction upload file format i.e. Fixed length. The Download Data feature enables you to either generate the specified system download file On Request or view a Pre-Generated download file.

The Download List screen allows you to view system downloads. The details are as displayed:

§        Process: Displays the download code for the specified file.

§        Description: Displays brief description of the specified download file.

§        On Request: Displays On Request hyperlink to view, generate and download data files 'on demand'.

§        Pregenerated: Displays PreGenerated hyperlink to view a previously generated download files.

Click the On Request hyperlink to go to the On Demand System Download screen to generate a download file on demand for a specific download process.

Click PreGenerated hyperlink, to go to the Pre Generated System Download screen to download pre-generated files of a specific download process.
