User Subscription Entry

The User Subscription Entry screen displays the Subscriptions that are available for the user. The details of the subscriptions are as listed below:

§        Subscription Code: View the code assigned to the Subscription.

§        Subscription Description: View the description assigned to the Subscription.

§        Subscription Conditions: The conditions under which the Subscription will be effective are listed under this column.

1.      Click the Details hyper link to view the details of a Subscription. The User Subscription Details screen appears showing the details of the event notification, the medium through which the notification is to be sent to the user and the condition of the notification. This will enable the client/bank user to identify the details of subscription before assignment.

2.      In the Action column, select the check box of the subscriptions that you want to assign to the User.

3.      Click Save to save the selection. This brings you back to the ‘User Subscription List’ screen where the newly assigned Subscription entry is displayed.

Note: These subscriptions are only available for viewing and cannot be modified.

OR

Click Back to return to the ‘User Subscription Entry’ screen without taking any action.
