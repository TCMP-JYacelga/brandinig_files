About the A/C Enrichment Master

You may specify the filter criteria to retrieve a particular Enrichment definition.

Defining a new Enrichment Entry:

§        Click Add Enrichments to go to the Enrichments Entry screen. The newly defined Enrichments entry is added to the Enrichment List, with its state specified as New and the same can be edited or discarded.

To enable the Enrichments, it is mandatory to authorize it based on the Maker-Checker concept. The checker can view the new definition in the Enrichments Authorization screen and authorize the same provided the rights to authorize are assigned to that checker.

A successfully authorized Enrichment definition request can be viewed in the Enrichment List screen with its status as Authorized.

Note: If you have created Enrichments that you do not want to enable for further use in the system and if those Enrichments are not yet authorized, you can Discard them.

Editing an Enrichment Entry:

1.      Click Edit to go to the Enrichment Entry screen where you can edit a previously defined master.

2.      Click Update to save the changes.

Note:

 You can modify the definitions if they are in any of the following states:

-         New : Any new definition is not yet authorized and is in the new state.

-         Modified : Any definition that you have modified and not yet authorized and is in the Modified state.

-         Authorized: The definition that is authorized and is in the Enabled state.

The edit option is not available for the definitions that have the state as Disabled, Disable Request, Enable Request, Disable Request Rejected, and Enable Request Rejected, Submitted, Authorized.

On saving the modifications, the definition state changes to Modified. To enable the definition with the Modified state, the definition has to be authorized.

 

Viewing the Preferred Account Name:

1.      Click View to go to the View Enrichments screen where you can view a previously defined master. You cannot modify the definition in this screen.

2.      Click Back to return to the Enrichments List screen.
