Initiating a Transaction using Template

Payments >> Payments >> Payments

1.      In the Payment center, select New From Template. The screen opens displaying all the existing templates.

2.      Select the template.

3.      Click the Clone  icon of the template that you wish to use to initiate a transaction.

4.      The Cloning facility is available only for those templates of Batch or Instrument (Quick Pay) with status as ‘Template Ready’. All the details entered in the Template will be copied onto the Batch and Instrument pages respectively.

5.      System will undertake the necessary validation to ensure that the initiated transaction is valid as per the current set rules. In case the template is saved with a registered beneficiary, system will copy the latest details of the beneficiary from the master and use it for transaction processing.

Note: For transaction initiated using templates, the Registered or Adhoc Beneficiary details will be non-editable.

6.      Click to save the transaction.

§        In case the transaction does not pass the system validation, the status of the Batch / Instrument will change to ‘For Repair’. You will have to repair the transaction before processing further.

§        Subsequent processing i.e. Submit / Verify / Authorize / Send is as per current system functionality
