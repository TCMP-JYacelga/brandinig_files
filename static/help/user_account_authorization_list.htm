User Account Authorization List

Admin >> User Setup Auth >> User Account Auth

To enable the User Account relation, it is mandatory to authorize the requests based on the Maker-Checker concept. The checker can view the new definitions in this screen and authorize the same provided the rights to authorize are assigned to that checker.

To authorize a User Accounts definition:

§        As Checker, select the Accept check-box to authorize the request.

You can filter the search by User to retrieve a particular definition.

§        You may also select the Reject check box to reject the request. While rejecting a request, it is mandatory to type the reason for rejection. Type the reject reason and click OK.

§        On successful authorization, the accounts can be viewed in the User Accounts List screen with the status as Authorized.
