Group Master Entry

The Group Master Entry screen allows you to set up a new Group definition.

1.      Specify the following Group details:

§        Group Code: Type a unique identification code for the group that you want to define.

§        Group Description: Type a description for the group code that you are defining.

§        Group Level: Type the group level. This is the hierarchal level of the group that is being defined.

§        Parent Group Code: Select the appropriate parent group under which the group that is being defined comes, from the drop-down list showing all the previously defined relevant parent groups.

Note: While defining the first Group, Group level should be 1. For the group with level 1 , the same group code will be defaulted as Parent Group Code.

2.      Click Save to save the new group. Once you have saved the entry, the new group is added to the Group List, with its status as New. This implies that this is a new addition and yet to be authorized.

To enable the Group, it is mandatory to authorize it based on the Maker-Checker concept. The checker can view the new definition in the Group Authorization screen and authorize the same provided the rights to authorize are assigned to that checker.
