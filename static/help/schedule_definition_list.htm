Schedule Definition List

Reports >> Reports >> Report Schedule Def

Scheduled Reports are reports that can be generated at a specified frequency, in a specific format. You can schedule a report to run repetitively on the desired medium. Once scheduled, these reports are automatically generated without any user intervention. You can view the details of schedules that have been defined for reports using the Reporting module

The Schedule Definition List screen displays all the scheduled reports in the system.

§        For each schedule, the following details are displayed: Description, Schedule, Media, Prev. Gen. Date. and Status.

§        You can filter the search by Report Code. The drop-down list shows all the report codes specified in the system.

Tip:  Click Discard to discard a new record or a definition status.

To define a new Schedule:

1.      Select a 'Report Code' from the drop-down list.

2.      Click Add to go to the Schedule Entry screen where you can enter the schedule details.

A newly defined schedule yet to be authorized, will appear in this screen with status as New and the same if needed can be discarded. On successful authorization in the Schedule Definition Authorization screen, the status will change to Authorized.

Viewing a Schedule Definition:

1.      Click View to go to the Schedule View screen where you can view a previously defined schedule. You cannot modify the same in this screen.

2.      Click Back to return to the Schedule Definition List screen.
