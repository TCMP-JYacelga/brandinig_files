Instrument Entry SI/ Quick Pay SI

1.      The Instrument Entry SI screen allows you to enter instrument-level transaction details of the Standing Instruction.

The fields will vary depending on the selected Product and are similar to the Payments Instrument Entry

The fields specific to Standing Instructions setup are as below and are required for setting up the frequency for executing the Standing Instruction.

□         SI Frequency: Select the frequency that you want to set up for the standing instructions from the drop-down list.

□         StartDate:Select the Effective Date from the calendar pop-up. The field gets populated with the start date in DD/MM/YYYY format, i.e. the date from when the frequency will be activated. Ensure that the Effective Date is greater than the application date.

□         End Date: Select the Termination Date from the calendar pop-up. The field gets populated with the end date in DD/MM/YYYY format i.e. the date till when the frequency will be valid. Ensure that the Termination date is always more than Effective Date. The default value for this field is- 'Till Terminated'.

□         No. of Times: Once you enter the SI Frequency, the Effective Date and the Termination Date this field automatically shows the number of times the payment will be processed.

□         Holiday Action: Select the action that should be taken in case the transaction date falls on a holiday as defined by the bank in the Holiday Setup.. The available options are- Postpone which is the default value, and Prepone.

For rest of the fields and further processing, refer to Payments- Instrument Entry.

Note:  Once the setup is saved, system will automatically initiate the SI transaction based on the specified frequency.

2.      Click to save the instrument details.

3.      In case advice information fields are pre defined for the Client Product combination, the Advice Info. tab is enabled and entering the details is mandatory. Click the Advice Info. tab to open the Standing Instruction Advice Info. screen click Add to add the values in the advice information fields defined for the product. Save the advice information and click Add to add more values.

Note: Every advice information needs to be saved before you can add the next one.

4.      Click Save to save the advice Information, and then click Back to exit from the Advice Info. entry screen to view the newly entered Advice Information in the Advice Info. summary screen.

5.      Click the Instrument tab to go to the 'Instrument' entry screen and click Save to save the instrument. A message indicating that the instrument is successfully saved appears on the screen. At this stage, you can delete the instrument by selecting it and clicking   that appears under More.

6.      In case of any errors, you can also Edit the instrument and click to update the transaction.  (To add more instruments, click 'Add Instrument'  in the Deposit header screen and then repeat the same process).

7.      Click Back to go to the Batch Header screen where you can view the newly added instrument under Instrument Details section. The Submit button is enabled which indicates that the transaction can be submitted for authorization. You also have the option to Discard it.

Note: If the instruction that you have entered is the last instruction within the payment transaction, system performs the checksum validation and notifies the discrepancies, if any. You need to correct these discrepancies before you close the payment transaction.

8.      Click the Back button till you go back to the Standing Instruction Center screen.
