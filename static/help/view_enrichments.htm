View Enrichments

The View Enrichments screen allows you to view previously defined enrichments pertaining to Balance Reporting, to an existing account or to all the accounts of a particular client.

Note: The enrichments defined in this master can be viewed in the Account >> Account Summary >> Account Activity >> View Enrichments

To View the Enrichments:

§        Enrichment Level: This indicates whether enrichments are applied at Account or Client  level.

In case you have select Account at 'Enrichment Level', view the Account Code as selected from the Account Seek list showing all the accounts of that client.

In case you have selected Client at 'Enrichment Level', this field is non-editable. The enrichments defined will apply to all the accounts held by that client.

§        Required: This indicates the enrichment that is required to be specified for transaction request to proceed further. i.e. the enrichment is mandatory for that transaction.

§        Screen Label: This indicates the enrichment field label as you want it to appear on the screen, e.g. Invoice No

Click Back to return to the Enrichments List screen.
