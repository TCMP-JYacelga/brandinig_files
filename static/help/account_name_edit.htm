Account Name Edit

The Account Name Edit screen allows you to define preferred names for the accounts that have been previously defined for balance reporting.

To define a Preferred Account Name:

§        Enter the details in the Account Name Edit screen.

□         Account Number: Displays the account number for which you want to define a preferred name.

□         Description: Displays the description for the selected account as specified in the Accounts master set up in the GCP CashWeb Admin Console module.

□         Account Usage: Type the purpose for which this account will be used. This is the preferred identifying name for the account.

□         Groups 1-5: Select the logical group that you want to link to this account. These are the groups that have been previously created in the Group Master.

□         Account Types 1-5: You can specify a preferred name to the selected group in this field.

□         Daily Receipt Limit: Select the check box in case you want to set the limit for daily Receipt.

□         Receipt Amount: This field is enabled only when you select the Daily Receipt Limit and allows you to specify the Receipt amount.

□         Daily Payout Limit: Select the check box in case you want to set the limit for daily Payout.

□         Payout Amount: This field is enabled only when you select the Daily Payout Limit and allows you to specify the Payout amount.

□         Low Book Balance: Type the lowest book balance that can be maintained in the account.

□         High Book Balance:Type the highest book balance permitted for the account.

□         Dr. Txn Amount: Type the maximum debit transaction amount allowed for the account.

□         Cr. Txn Amount:Type the maximum credit transaction amount allowed for the account.

□         Dr. Txn Count: Type the maximum debit transactions  allowed for the account

□         Cr. Txn Count: Type the maximum credit transactions allowed for the account.

§        Click Update to go back to the Account Name List screen to view the newly named account, with its status as Modified.

Click Back to go to the previous page without editing the account name.
